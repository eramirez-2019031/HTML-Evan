Proyectos Desarollados

  • Tonys Kinal

    Este proyecto es de los mas recientes, el cual fue hecho con la intencion de crear una tienda de abarrotes, fue creado totalmente en java, usando los conocimientos aprendidos en 5to año, tardo aproximadamente unos 4 meses, posee muchos controladores y esta enfocado en ser un inventario de esta tienda, llevando el conteo de un base de datos creada en MySQL, junto a un login y diseño personalizado.

  • Proyecto de Ventas

    A difertencia de mi proyecto anterior, este fue desarrolado con html en Java, no teniendo ninguna vista hecha en algun programa externo, las vistas fueron desarrolladas por medio de codigo, este se trataba principalmente de un programa de ventas virtuales, contando con su base de datos propia, junto a un login y un diseño minimalista, contando incluso con imagenes de cada usuario y un carrito de compras para dicho invenrtario

  • eBook Hub

    Este proyecto fue creado de dorma simimlar a la de tonys kinal, siendo esta completamente en java, lo especial de este trabajo es que fue nuestro primer proyecto realizado con la metodologia Scrum, ademas que al ser muchos integrantes el trabajo fue considerablemente mas grande que los anteriores, este se trata de una libreria virtual, donde la gente puede comprar sus libros de forma fisica y que sean enviados, o comprar una suscripcion para poder leer libros de forma virtual, contando con sus propios generos de literatura, y poder ver las descripciones de cada libro.
